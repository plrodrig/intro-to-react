# Introduction to React
A 3-day React and Redux course for professional developers.

# Setup and Testing Your Setup
1. Install a recent version of node.js
1. ```> git clone https://github.com/watzthisco/intro-to-react```
1. `> cd intro-to-react`
1. `> npm install`
1. `> npm test`
